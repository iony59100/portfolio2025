Application web en React pour l'hôpital


Description du projet


Il s'agit d'une interface web moderne pour permettre aux utilisateurs de gérer les rendez-vous à l’hôpital, consulter des informations et naviguer de manière fluide.


Le déroulement du projet


-Conception de l’arborescence de l’application

-Création des composants React

-Utilisation du router pour la navigation

-Connexion avec une API pour récupérer les données

-Gestion des états avec React (et/ou Redux selon le projet)

-Tests, améliorations visuelles et responsive design

Compétences mises en œuvre :

-Développer la présence en ligne de l'organisation

-Mettre à disposition des utilisateurs un service informatique

-Développer des applications web dynamiques avec React

-Travailler en autonomie sur une interface utilisateur
